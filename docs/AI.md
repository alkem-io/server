# Useful Prompts

## Running Speckit tasks for 1 token!

```text
This branch contains a detailed specifications for a <your user story>, made using specification driven development.
Your task is to execute the final stage, following the execute prompt in the .github/prompts folder.
Do not change the specifications that can be founds in the specs folder at the root of the repo, but implement all phases of the project as specified in the tasks.md.
If there is any ambiguity in any of the specifications, don't stop the coding, but make reasonable assumptions.
Finish all stages, don't stop until everything is complete!
This is a complex task - use your best available model.
```
