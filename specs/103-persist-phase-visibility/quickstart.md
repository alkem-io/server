# Quickstart: Persist Phase/Tab Visibility

## What this delivers

`visible: Boolean!` on `InnovationFlowStateSettings` (default `true`), togglable by admins via the
existing `updateInnovationFlowState` mutation, persisted in the JSONB `settings` of each
`innovation_flow_state`, with a backfill so existing phases stay visible. Visibility is a UI hint
only — it never gates content access.
