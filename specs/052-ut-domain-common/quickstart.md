# Quickstart: Unit Tests for src/domain/common

## Run all domain/common tests
```bash
pnpm test src/domain/common
```

## Run a specific test file
```bash
pnpm test src/domain/common/nvp/nvp.factory.spec.ts
```

## Run with coverage
```bash
pnpm test -- --coverage src/domain/common
```

## Lint check
```bash
pnpm lint
```

## Type check
```bash
pnpm exec tsc --noEmit
```
